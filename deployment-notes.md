## Deployment Options

### Option 1: GitHub Pages (Recommended - Free)
1. Create a GitHub repo
2. Upload landing-page.html as index.html
3. Enable GitHub Pages in settings
4. Get URL like https://yourusername.github.io/garage-cleanout

### Option 2: Netlify Drop (Free - Instant)
1. Go to https://app.netlify.com/drop
2. Drag and drop the landing-page.html file
3. Get instant URL (like https://bright-cactus-123.netlify.app)
4. Can connect custom domain later

### Option 3: Tiiny.host (Free - Simple)
1. Go to https://tiiny.host/
2. Upload the HTML file
3. Get URL like https://garage-cleanout.tiiny.site

### Option 4: Carrd (Requires Manual Build)
- Carrd free plan doesn't allow HTML import
- Would need to manually rebuild using their editor
- This is time-consuming via browser automation

## Recommendation
Given the Facebook App Review deadline, I recommend **Netlify Drop**:
- No signup required
- Instant deployment
- Free subdomain
- Easy to update later
